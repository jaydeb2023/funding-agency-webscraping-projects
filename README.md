echo "# Global Funding Agency Web Scraping Projects
This repository contains Python-based scraping scripts for multiple international funding agencies: AFD, CAF, CDB, EIB, and AIIB.
Each script extracts project and funding data using Selenium, Requests, and BeautifulSoup for analysis.

## Agencies Covered
- AFD (Agence Française de Développement)
- CAF (Development Bank of Latin America)
- CDB (Caribbean Development Bank)
- EIB (European Investment Bank)
- AIIB (Asian Infrastructure Investment Bank)

## Tools & Libraries
- Python
- BeautifulSoup
- Selenium
- Pandas
